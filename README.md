## AIChaBot_NLP
NLP Chatbot using NLTK

A simple rule-based chatbot built using the Natural Language Toolkit (NLTK) library in Python.
This chatbot uses pattern matching and predefined responses to simulate conversation.

## Project Description

This project demonstrates:
1. Basic Natural Language Processing concepts
2. Pattern matching using regular expressions
3. Rule-based conversation handling
4. Use of nltk.chat.util.Chat
5. Reflections for natural responses

The chatbot responds to greetings, introductions, apologies, and general input.

# Technologies Used
1. Python 3
2. NLTK Library
3. Regular Expressions (re)

Installation & Setup:
1. Install Python:
   Make sure Python 3 is installed on your system.
2. Check version:
   python --version
3. Install NLTK
Run the following command:
   pip install nltk

## How to Run the Chatbot

Navigate to the project directory and run:

   python chatbot.py
